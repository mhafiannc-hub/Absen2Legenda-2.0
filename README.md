# Absen2Legenda

Project ini sudah disusun ulang menjadi aplikasi React + Vite.

## File utama
- `index.html`: entry HTML
- `src/main.jsx`: bootstrap React
- `src/App.jsx`: aplikasi absensi utama
- `src/index.css`: styling dasar
- `.env.example`: contoh konfigurasi Firebase

## Cara jalan
1. Salin `.env.example` menjadi `.env`
2. Isi semua kredensial Firebase
3. Jalankan `npm install`
4. Jalankan `npm run dev`

## Catatan
- `script.js` dan `style.css` lama masih dibiarkan di folder sebagai versi lama dan saat ini tidak dipakai oleh entry React baru.
- Login admin memakai email yang diatur di `VITE_ADMIN_EMAIL`.
